# EZMR

Die Dokumentaion erläuert die Konzepte der Programmieraufgaben und beschreibt die Funktion der entwickleten Anwendungen. Weiterhin werden die Grundkonzepte von ROS2 und Versuche mit einem Testroboter dargestellt. Abschließend werden die erarbeiteten Konzepte an einem profesionellen Turtlebot4 getestet.

+ Der Ordner minibot enthält alle Datein und Videos zum Testroboter.
+ Der Ordner turtlebot4 enthälz alle Datein, Fotos und Videos zum Turtlebot4.
